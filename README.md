# Community Care Connect

Static patient-facing self-referral portal for Gravity Project SDOH Connectathon Scenario 1: Individual Self-Referral.

## Files

- `index.html` - page structure and self-referral form
- `styles.css` - responsive teal and blue design
- `app.js` - service data, category filtering, modal behavior, and confirmation state

## Render Static Site Settings

- Build command: leave blank
- Publish directory: `.`

The site does not require a backend or framework.
